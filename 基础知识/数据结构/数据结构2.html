 树

  定义：n个节点的有限集合，n=0时为空树，在任意一个非空树有以下定义

   每一个树都有且仅有一个节点被称为根

   n>1其中每个子集本身又是一棵树，被称为子树

  节点的层次

   节点的最大层次为根的深度或高度

  节点的度

   节点拥有的子树的数量被称为度

  父子节点

   节点派生出的节点称为

  兄弟节点和堂兄弟节点

   兄弟节点

    同一个父节点

   堂兄弟节点

    同一个爷爷节点，父节点不同

  有序树

   节点的各子树看成是从左到右是有序的，不能互换的

  二叉树

   定义：每个节点最多只有两个子树（度最大为2），子树有左右之分，且左右不能颠倒

   斜树

    左斜树

    右斜树

   满二叉树

    每一层的节点数都是最大节点数（除了最底层）

   完全二叉树

    满二叉树的子集，自上而下，自左而右进行编号，编号相同的节点与满二叉树相同编号的节点相同

    与满二叉树相比只能缺少叶子节点，而且只能从右到左地缺少

   二叉搜索树（bstree）

    二叉搜索树定义

      任意节点的左子树不空，则左子树上所有结点的值均小于它的根结点的值；

      任意节点的右子树不空，则右子树上所有结点的值均大于它的根结点的值;

      任意节点的左、右子树也分别为二叉搜索树；

      一般没有键值相等的节点。

      规律

       前序遍历第一个访问的一定是整个二叉树的根节点

       中序遍历根节点前访问的节点一定在根节点的左子树中，根节点后访问的节点一定在根节点的右子树中

       后序遍历中根节点一定是最后被访问到的

       主要思想是递归，以前序遍历举例，在二叉树中，先输出根节点的数值，之后以前序遍历对左子树进行输出，左子树输出完毕后，对右子树进行输出，在每个子树处也执行前序遍历

  节点的高度如何求

   叶子类的节点高度为1，子树根的深度是其左右子树高度较大者+1

   使用递归求一树的高度（参数为树的根节点）

     int avltree::height(PNODE & node) { // 空树返回 0 //叶子节点深度为1 if (node == nullptr) return 1; // 获取左子树的高度，右子树的高度 int left = height(node->left); int right = height(node->right); // 返回当前结点的高度 return left > right ? left + 1 : right + 1; }
