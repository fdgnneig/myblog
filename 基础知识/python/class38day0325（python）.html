 不可变类型（指同一个内存空间中的值不能改变）

  int string float tuple

  不可变数据类型： 当该数据类型的对应变量的值发生了改变，那么它对应的内存地址也会发生改变，对于这种数据类型，就称不可变数据类型。

  number =1 print(hex(id(number))) number =2 print(hex(id(number)))

   number类似于一个指针，被不同值赋值，指向不同的变量

   内存空间中的值不被改变，但此类型数据类似与指针，其指向的空间不能发生变化，但是指向可以变化

  元祖中的每一个元素、每一个int srting变量均可以看成是一个指针，指向静态内存地址，指针指向可变，内存地址值不变

  如果是元组是不可变数据类型，其中元素值如果发生变化，则其内存地址会发生变化，

   当元组中的元素为列表时，列表是可变类型，改变元祖中列表中的元素值，列表的内存地址不会发生变化，这是也由于列表本身的可变类型决定的，列表在元组中的地址的值没有改变，所以也就意味着元组没有发生变化。
