 访问控制权限

  单下划线_代表模块级私有

   在模块中以_开头的变量、函数为模块私有，无法通过 from 模块名 import*导入其他程序使用

   但是仍可以通过 import 模块名 模块名.函数名(参数) 的方式导入

  双下划线实现一定程度上的私有

   class obj() def _init_(self,name): self.__name=name b=obj("小明") b.__name 无法访问

    此时通过 b.__name访问不到

    通过某种方法是可以访问到的

  约定俗称在类中添加单下滑线，表示该变量或方法私有，虽然可以访问，但是请把该变量当成私有

  双下划线开头且结尾的变量可以访问，但是多为系统定义且存在特殊意义，一般不要定义此类数据成员

  可以单继承，可以多继承

    可以多继承，则会出现菱形继承的问题，在孙子类访问爷爷类的成员数据时，孙子类先继承哪个父类，就是使用该父类中继承自爷爷类的数据成员

     class Base(object): def __init__(self,name): self.name = name class A(Base): def __init__(self): Base.__init__(self,"我是A") class B(Base): def __init__(self): Base.__init__(self,"我是B") class C(B,A): pass obj = C() print(obj.name) #如果B继承在前，则输出的是"我是B"，否则输出"我是A"

异常

 python允许程序编写者得到异常并处理，使得程序还能够继续执行

 如果知道异常类型，可以直接捕获异常，并进行异常处理，程序不会停止

  try： print(a) except NameError: pass#这里针对异常不做处理，直接跳过 print("15pb")

   这里错误异常的种类是IndecError，检测到的异常与指定的异常不符，不能执行异常处理程序，程序运行停止

   主动抛出异常之后可以进行捕获，并输出指定的错误提示信息

特殊函数

 filter(过滤器)

  filter() 函数用于过滤序列，过滤掉不符合条件的元素，返回由符合条件元素组成的新列表。 该接收两个参数，第一个为函数，第二个为序列，序列的每个元素作为参数传递给函数进行判，然后返回 True 或 False，最后将返回 True 的元素放到新列表中。

  filter(funcname,list)

   第一个参数为函数名，该函数为回调函数，参数只有一个，返回值为BOOL

   第二个参数为一个序列，可以使用列表生成式传参

    [x for x in range(50)]

   该函数会对序列中的项依次执行回调函数，并将结果为true的组成一个序列返回

   def filter_fun(arg): return arg%2==0 L = list(filter(filter_fun,[x for x in range(50)])) print(L)

    取得0-49中所有的偶数的数值作为一个list

 map

  map(funcname,list)

   map() 会根据提供的函数对指定序列做映射。 第一个参数 function 以参数序列中的每一个元素调用 function 函数，返回包含每次 function 函数返回值的新列表。

   回调函数参数只有一个，返回值可以为任意类型

   python中只有字符串没有字符，则数字与字符之间的转换可以使用相关函数

   对序列中的项依次执行回调函数，执行结果组成一个list返回，map支持多序列

    与filter的区别在于map可以返回所有经回调函数处理过的结果，而filter只能返回函数执行结果为true的序列项

   def map_fun(arg): return arg.lower() #转换为小写，upper()为转换为大写 L=list（map(map_fun,[chr(x) for x in range(65,91)])） print(L)

    将数转为大写字母，在map_fun函数中将大写字母转为i小写字母

 reduce

  reduce() 函数会对参数序列中元素进行累积。 函数将一个数据集合（链表，元组等）中的所有数据进行下列操作：使用传给 reduce 中的函数 function（有两个参数）先对集合中的第 1、2 个元素进行操作，得到的结果再与第三个数据用 function 函数运算，最后得到一个结果。

  from functools import*

   需要导入模块

  类似于数组或列表

  reduce(funname,list)

   回调函数参数有两个，返回一个整数

  reduce会从序列中取出两个数进行回调函数计算，得到的结果再从序列中取出下一个带入回调函数

  from functools import* def reduce_fun(arg1,arg2): return arg1+arg2 number=reduce(reduce_fun,[x for x in range(1,101)])

   从1加到100

 装饰器

  作用：装饰器的作用就是为已经存在的函数或对象添加额外的功能。

   外层函数返回的是里层函数的函数指针，即内层函数本身被返回

  def w1(old_fun): def inner(arg1,arg2): # 校验 if(type(arg1)!=int): raise NameError("不是整形")#为old_fun函数提供数据合法性检验 # 调用旧函数 return old_fun(arg1,arg2) #最后调用old_fun return inner @w1 #这个语法糖的意义在于 fun=w1(fun) w1函数将其内部函数返回回来，其内部函数除了会执行fun函数，进行数据合法性检验 #则在该函数执行时，可以运行装饰器为函数添加额外的数据合法性检验的功能 def fun(arg1,arg2): return arg1+arg2 print(fun("hello","15pb"))#执行之后，数据非法，会抛出错误
