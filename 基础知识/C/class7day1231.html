结构体和联合体

 结构体

  数组用于保存多个相同类型的数据

  多个不同类型的数据使用结构体进行保存处理

  结构体类型是一个复合数据类型,不能参与运算

  结构体中的成员,也可以叫做字段

  结构体的声明一般放在main函数外面,并且在第一次使用之前

  两个结构体变量之间是可以赋值的

  结构体类型名首字母单词大写

  结构体定义

   struct student { int nId; int num; char name[10]; };

   结构体使用

    一般先定义结构体类型,再定义结构体变量

    一般在纯c的环境下定义结构体变量的时候 需要加上关键字struct,但是在c++的环境下struct可以省略

     struct 结构体类型名 结构体变量名;

      struct student xiaoming={1,2,"xiaoming"};

    访问结构体内部成员的时候使用.

  结构体变量的大小

   结构体声明不会占用内存

    sturct person { char name[10]; int number; };

     typedef struct person { char name[10]; int number; }per,*pper;

     直接将结构体类型的名称改成per方便之后使用

     这里也顺便定义了该种结构体类型的指针 此时定义指针就不需要加* 即pper p=NULL;定义了一个结构体类型的指针

   结构体变量的大小取决于其内部成员数据类型大小之和,如果成员之和不满足4的倍数,系统会自动补齐,使结构体变量占有的内存为四的倍数

    即存在内存对齐机制

  结构体变量的初始化

   sturct person { char name[10]; int number; }; sturct person stcper{"15pb",1};

   部分初始化

    int main() { struct person obj={0}; }

     全部初始化为0

    嵌套初始化中,作为成员的结构体可以使用{}在初始化的时候进行初始化{{1,2},3,4,5};

   完全初始化

  结构体变量的访问

   使用.访问结构体变量中的成员(即字段)

   .操作数 左边操作数是结构体变量 右操作数是结构体字段,运算结果是结构体字段的值,类型是字段的类型

  结构体内存空间

   结构体在内存中以字段先后为顺序,依次对各个字段进行储存

  结构体指针

   struct Mystruct*pObj=&obj1

   使用结构体指针引用结构体的字段

    pObj->结构体的字段

    (*pObj).结构体的字段

   当使用结构体指针作为函数参数进行按地址传递参数时,如果不想在被引用函数中改变传入的实参,可以在参数前面加上const,将形参指定为常量指针,不能改变该指针的指向

    void fun(const struct Mystruct*p);

     当const在*的左边时,值指的是指针指向的数据被指定为const类型,即指针指向的值是不可以修改的(但是此时指针可以指向别的数据)

    void fun(struct Mystruct*const p)

     const在*右边是指const修饰的是指针,说明指针的指向不可以修改,但是指向的数据是可以修改的

  对于专门服务于结构体的函数命名的时候可以有相应的格式

  一般在头文件中声明结构体,然后找一个同名的.c文件存放结构体的定义并在该.c文件中#include<前面的.h文件>

   然后其他文件只要包含前面的头文件,既可以使用该结构体(全局变量)

  scanf_s("%d",&((student+i)->id)); //注意在结构体数组的引用中,因为要使用. 或-> 而这两者因为最终得出的都是对应成员变量的值, //所以在表示字段的地址时一定会使用&,而不可能直接使用数组名称作为指针直接表示出成员变量的值
